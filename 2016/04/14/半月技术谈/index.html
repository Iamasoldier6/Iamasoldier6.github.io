Java

博主前段时间深感 Java 基础无力，于是自清明放假开始，重新撸起去年自学期间半途放弃的《Thinking in Java》，并打算一季左右撸完。目前，已撸到了第 7 章，正常进度应该可以达成初定目标吧，原谅我是个渣渣，大神勿喷。几点总结如下：

1.在 Java 中你所做的全部工作就是定义类，产生那些类的对象，以及发送消息给这些对象。

2.一个 static 字段对每个类来说都只有一份存储空间，而非 static 字段则是对每个对象有一个存储空间。

3.构造器的名称必须与类名完全相同。System.gc() 用于强制进行终结动作。

4.通常写 this 的时候，都是指“这个对象”或者“当前对象”，而且它本身表示对当前对象的引用。

5.初始化的顺序是先静态对象，而后是“非静态”对象。

6.重构即重写代码，以使得它更可读、更易理解，并因此而更具有可维护性。

7.关于public, protected, default 与 private 的作用域：

作用域 当前类 同一包内 子类 其他包
public √ √ √ √
protected √ √ √ X
default √ √ X X
private √ X X X

Android

1.Android Studio 的中文乱码问题：Android Studio 安装后发现所有的中文，不管是界面上的还是输出的 Log 中的中文，都变成问号小框框， 可以肯定是字体的问题。

解决办法：Preferences -> Appearance & Behavior -> Appearance -> 选择 Override default fonts by，随便选择一个中文字体，小框框都是中文字，再保存重启就行。

2.Android Studio 小技巧合集，熟练一些常用快捷键操作，简直不能太顺心。
